# webclient

The React frontend for Sentinel Finance. It progressively streams stock search results, renders interactive price charts with article event pins, and manages user watchlists — all backed by a typed API client generated from the OpenAPI contract.

---

## What this service does

- **Home page** — search bar + trending tickers grid loaded on mount.
- **Search results** — live-streaming result cards that appear as each stock is scored; a three-stage pipeline (tickers → articles → sentiment) updates each card progressively.
- **Stock detail** — OHLC price chart with time-range selector (Today, 1D, 1W, 1M, 1Y), detected price events pinned on the chart, article list that highlights the window for the hovered event, a competitor accordion with peer sentiment, and an optional LLM insight card.
- **Watchlist** — authenticated users can create named lists, add/remove tickers, and receive real-time notifications when sentiment diverges for a watched ticker.
- **Auth** — register/login modal backed by a JWT token stored in local state.

## Local development

### Prerequisites

- Node.js 22 LTS, pnpm 10
- `core` running on `http://localhost:3001`

### Setup

```bash
cd webclient
pnpm install
pnpm run dev      # Vite dev server on http://localhost:5173
```

API calls are proxied to `http://localhost:3001` via the Vite config. No separate env file is needed for local development.

### Available scripts

| Script                        | Description                                        |
| ----------------------------- | -------------------------------------------------- |
| `pnpm run dev`                | Start Vite dev server with HMR                     |
| `pnpm run build`              | TypeScript compile + Vite production build         |
| `pnpm run preview`            | Preview the production build locally               |
| `pnpm run typecheck`          | Type-check without emitting                        |
| `pnpm run lint`               | ESLint                                             |
| `pnpm run format`             | Prettier                                           |
| `pnpm run contracts:generate` | Regenerate API client from `contracts/openapi.yml` |

│   ├── ResultCard.tsx        Stock card used in search results and trending grid
│   ├── SentimentInsight.tsx  LLM insight card
│   └── StockTimeline.tsx     OHLC chart with event pins (Recharts)

├── context/                  React context providers

│   └── watchlist-provider.tsx    User list state and mutations
│
├── hooks/                    Data-fetching hooks

│   ├── useTickerPipeline.ts  Latest-mode and event-mode article+sentiment streams

│   ├── useCandles.ts         OHLC candle fetch with caching

│   └── useNdjsonStream.ts    Low-level NDJSON stream hook
│
├── pages/
│   ├── Home.tsx              Landing page with trending grid

│   ├── StockDetail.tsx       Detailed stock view with chart and articles

│

└── models/                   TypeScript model types (Article, Stock, InvestmentInsight, …)

## API client codegen

The API client in `src/api/generated/` is generated by [orval](https://orval.dev/) from `contracts/openapi.yml`. Regenerate it after any contract change:

```bash
pnpm run contracts:generate
```

Never edit the generated files — changes will be overwritten on the next codegen run.

## Adding UI components

This project uses [shadcn/ui](https://ui.shadcn.com/). To add a new component:

```bash
npx shadcn@latest add <component-name>
```

Components are placed in `src/components/ui/` and can be imported as:

```tsx
import { Button } from "@/components/ui/button"
```
